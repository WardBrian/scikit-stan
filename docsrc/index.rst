:mod:`sk-stan-regression` - Stan Regression Models in sk-learn Style
====================================================================

SciKit Stan Regression is a Python package of generalized linear models in the Stan with the familiar SciKit-learn interface. This package is a thin wrapper around the CmdStanPy package, which is itself a wrapper around the CmdStan command line interface.
